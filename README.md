# Cash Loan Management System

## 📌 Project Overview
The Cash Loan Management System is a web-based application designed to simplify the process of applying for, managing, and repaying cash loans. The system supports borrowers, loan officers, and administrators across multiple branches.

## 🚀 Features
- User registration and secure login
- Borrower loan applications
- Multi-branch support
- Loan approval and management by officers
- Payment tracking and loan repayment
- Branch-based reporting and analytics
- Document uploads for loan verification

## 🛠️ Technology Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js + Express
- Database: MySQL
- Authentication: JWT
- Payments: Payshap (planned integration)
